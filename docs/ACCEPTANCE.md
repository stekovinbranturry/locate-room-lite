# LocateRoom Lite — 验收清单

> 对照 [需求说明](./REQUIREMENTS.md) 逐项自测与提交前检查。  
> 每项对应的测试方式（Vitest / Playwright / 人工）见 [测试说明](./TESTING.md)。

## 功能验收

### 房间

- [ ] 可创建房间并生成可分享的链接
- [ ] 他人通过链接可加入同一房间
- [ ] 页面展示当前在线成员列表
- [ ] 成员加入时有实时通知（UI 或事件可感知）
- [ ] 成员离开时有实时通知（UI 或事件可感知）

### 位置共享

- [ ] 每人位置上报频率达到 **10 Hz**（可测量或 README 说明测量方法）
- [ ] 所有成员位置在地图上实时展示
- [ ] 位置数据经 **WebRTC DataChannel P2P** 传输（README 中说明拓扑与验证方式）

### 稳定性

- [ ] 新成员加入后，**立即**能看到房间内已有成员的最新位置（无需等待下一轮全量同步）
- [ ] 短暂断网后连接/同步可**自动恢复**
- [ ] 其他成员掉线时，当前用户能**明确感知**（列表、地图标记或状态提示）

## 性能验收

- [ ] **4 人**同房间同时在线，功能正常
- [ ] 4 人房间下位置更新维持 **10 Hz**（README 附测量数据或方法）
- [ ] 地图/位置渲染帧率 **≥ 50 fps**（README 附测量数据或方法）
- [ ] 从加入房间到**首次看到他人位置** ≤ **3 秒**（README 附测量数据或方法）
- [ ] **H5** 在中端 Android 浏览器可用
- [ ] **H5** 在 iOS Safari 可用
